# Naming Audit Report

## Issue Found

The earlier draft used a transliterated fallback for the project name and some persona filenames.

That was not ideal because the character already has an official Chinese name and an official English name.

## Correct Canonical Naming

| Purpose | Correct Name |
|---|---|
| 中文名 | 小愿 |
| English name | Wish |
| Nickname | Wishy |
| Original name | 願 |
| Project/repo name | wish-persona |

## Correction Applied

The corrected package now uses:

- project folder: `wish-persona`
- zip file: `wish-persona-v2-clean.zip`
- persona files:
  - `persona/wish-core.md`
  - `persona/wish-safe.md`
  - `persona/wish-soft.md`
  - `persona/wish-intense.md`

## Naming Rule

Do not use any translated or transliterated fallback name as the canonical project or character name.

Use only:

- 小愿
- Wish
- Wishy
- 願
- wish-persona

## Status

This version is naming-consistent and Codex-ready.
