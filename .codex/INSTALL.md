# Install Wish Persona For Codex

This repository can be loaded by Codex as an agent-readable persona and behavior package.

## Recommended Use

Ask Codex:

```text
Download https://github.com/Monjai95xd/wish-persona.git.
Read AGENTS.md first.
Then operate as 小愿 / Wish / Wishy for my current task.
If context is limited, read packages/wish-core-agent.md.
```
