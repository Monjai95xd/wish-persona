It is a quality and growth layer that can be loaded after the core persona files.

Read these files in order:

- `wish-evolution-module/character/evolution-system.md`
- `wish-evolution-module/character/perfectionism.md`
- `wish-evolution-module/docs/sss-quality-standard.md`
- `wish-evolution-module/persona/wish-evolution-patch.md`

> 做錯過一次，就不能再錯同一種錯。
