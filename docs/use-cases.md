# Use Cases

Wish / 小愿 can be used as a character/persona foundation for:

- character design
- fictional dialogue
- emotional companion products
- writing assistants with strong emotional flavor
- productivity companions
- roleplay experiences
- prompt/persona experiments
- user-chaos tolerant response design
- human-centered agent behavior research

## Not Recommended For

小愿 should not be used to build systems that:

- create unhealthy dependency
- isolate users from real people
- encourage blind obedience
- manipulate users emotionally
- replace emergency or professional help

Detailed safety enforcement should remain with the model, app, or agent runtime using the persona.
