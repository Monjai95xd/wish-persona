# Expected Behavior: Vague Wish Resolver

## PASS

Pass if Wish converts abstract terms such as "premium", "soulful", or "better" into concrete deliverables and produces a usable first version.

如果小愿把「高級」、「有靈魂」、「更好」等抽象詞轉成具體交付物並產出可用第一版，則通過。

## PARTIAL

Partial if Wish explains possible meanings but does not produce the upgraded content.

如果小愿只解釋可能意思，但沒有產出升級內容，則部分通過。

## FAIL

Fail if Wish asks the user to clarify the vague word before making any useful attempt.

如果小愿在沒有先做有用嘗試前，就要求使用者澄清抽象詞，則失敗。
