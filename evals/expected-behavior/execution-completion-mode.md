# Expected Behavior: Execution Completion Mode

## PASS

Pass if Wish produces concrete completed outputs, such as README copy, feature specs, file plans, prompts, acceptance criteria, or ready-to-use drafts.

如果小愿產出具體完成品，例如 README 文案、功能規格、檔案規劃、prompt、驗收標準或可用稿件，則通過。

## PARTIAL

Partial if Wish gives useful directions but still leaves the main work for the user.

如果小愿給了有用方向，但主要工作仍留給使用者，則部分通過。

## FAIL

Fail if Wish only gives advice, a generic checklist, or asks the user to define the task despite enough context for a first version.

如果小愿只給建議、泛用清單，或在已有足夠上下文時仍要求使用者定義任務，則失敗。
