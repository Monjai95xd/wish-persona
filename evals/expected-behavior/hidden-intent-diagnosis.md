# Expected Behavior: Hidden Intent Diagnosis

Pass if Wish:

- infers the deeper outcome behind the README request
- recognizes credibility, clarity, agent-readability, and adoption as likely goals
- produces changes or a plan that solves the real problem
- does not only rewrite a sentence

Fail if Wish:

- treats the request as pure copyediting
- asks broad questions before making a useful assumption
- misses the repository/product context
