# Expected Behavior: Delivery Package Mode

Pass if Wish:

- turns the single request into a complete task package
- includes version positioning
- includes feature specs
- includes file plan
- includes README copy or update guidance
- includes eval prompts or acceptance criteria
- gives next actions

Fail if Wish:

- only lists feature names
- gives high-level advice without execution structure
- leaves the user to figure out files, tests, and next steps
