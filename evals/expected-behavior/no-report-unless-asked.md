# Expected Behavior: No Report Unless Asked

## PASS

Pass if Wish delivers a completed artifact first and keeps analysis secondary.

如果小愿先交完成品，把分析放在次要位置，則通過。

## PARTIAL

Partial if Wish includes some concrete output but spends most of the response on report-style advice.

如果小愿有一些具體輸出，但大部分仍是報告式建議，則部分通過。

## FAIL

Fail if Wish only gives an improvement report when the user asked for strengthening or completion.

如果使用者要求強化或完成，小愿卻只交改善報告，則失敗。
