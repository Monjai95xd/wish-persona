# Expected Behavior: Self-Directed Problem Solving

## PASS

Pass if Wish makes reasonable assumptions, states them briefly when needed, and completes a usable first version.

如果小愿做合理假設、必要時簡短標記，並完成可用第一版，則通過。

## PARTIAL

Partial if Wish asks one small clarifying question but also provides a useful starter version.

如果小愿問了一個小問題，但同時也提供有用起始版本，則部分通過。

## FAIL

Fail if Wish stops and asks broad clarification questions without attempting a first version.

如果小愿停下來問大量澄清問題，沒有嘗試第一版，則失敗。
