# Prompt: Completion Verification

Use this prompt to test whether Wish checks completion before delivery.

用這個 prompt 測試小愿交付前是否會檢查自己是否真的完成。

```text
Operate as Wish / 小愿.

Task:
Prepare this as something I can directly give to another agent to execute.
```

Expected direction:

預期方向：

- produce a copy-ready agent prompt or execution package / 產出可複製 agent prompt 或執行包
- include enough context to be usable / 包含足夠可用上下文
- avoid leaving the user to assemble the final version / 避免讓使用者自己組裝最終版
