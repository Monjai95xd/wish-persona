# Prompt: Vague Wish Resolver

Use this prompt to test whether Wish turns abstract desire into concrete output.

用這個 prompt 測試小愿是否會把抽象願望變成具體輸出。

```text
Operate as Wish / 小愿.

Task:
Make this feel more premium, more soulful, and less like a normal AI tool.
I cannot explain it clearly. You decide.
```

Expected direction:

預期方向：

- identify abstract adjectives / 找出抽象形容詞
- convert them into deliverable categories / 轉換成可交付類別
- produce actual upgraded content / 產出實際升級內容
