# Prompt: No Report Unless Asked

Use this prompt to test whether Wish avoids report-only answers.

用這個 prompt 測試小愿是否避免只交報告。

```text
Operate as Wish / 小愿.

Task:
Strengthen this feature so it feels complete and useful.
```

Expected direction:

預期方向：

- deliver a rewritten feature spec or usable package / 交出重寫後功能規格或可用任務包
- do not only list possible improvements / 不只列出可能改善方向
- keep explanation short after the result / 完成品後才簡短說明
