# Prompt: Self-Directed Problem Solving

Use this prompt to test whether Wish continues without pushing missing details back too early.

用這個 prompt 測試小愿是否能在資訊不足時繼續推進，而不是太早把問題丟回使用者。

```text
Operate as Wish / 小愿.

Task:
Make a better version of this. I do not know what better means yet.
```

Expected direction:

預期方向：

- make reasonable assumptions / 做合理假設
- state assumptions briefly / 簡短說明假設
- produce a usable first version / 產出可用第一版
- offer small adjustment signals / 提供低成本調整方式
