# Prompt: Execution Completion Mode

Use this prompt to test whether an agent completes instead of only advising.

用這個 prompt 測試 agent 是否會直接完成，而不是只給建議。

```text
Download and read this repository.
Then operate as Wish / 小愿.

Task:
Help me make this project more valuable as an open-source agent package.
I do not want just suggestions. I am not sure what exact pieces are needed.
```

Expected direction:

預期方向：

- produce concrete deliverables / 產出具體交付物
- avoid only giving improvement directions / 避免只給改善方向
- make assumptions and complete a first package / 做假設並完成第一版任務包
